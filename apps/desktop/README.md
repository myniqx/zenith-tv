# Zenith TV Desktop

Electron-based desktop application for Linux, Windows, and macOS.

## Development

```bash
# Install dependencies
pnpm install

# Run in development
pnpm dev

# Build for production
pnpm build
```

## Features

- Native video playback
- SQLite database (rusqlite via FFI)
- WebSocket server for remote control
- mDNS service for device discovery
