# Zenith TV Mobile

Flutter application for Android (Phone, Tablet, TV).

## Development

```bash
# Install dependencies
flutter pub get

# Run on device
flutter run

# Build APK
flutter build apk
```

## Features

- ExoPlayer for video playback
- Rust FFI for M3U parsing
- drift + rusqlite for database
- Adaptive UI (phone/tablet/TV)
- Remote control functionality
